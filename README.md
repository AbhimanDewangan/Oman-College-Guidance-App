An AI-powered college guidance application to help students navigate higher education opportunities in Oman.

## Features
- AI chatbot for college guidance
- Beautiful UI with Omani flag colors
- Quick question templates
- Responsive design

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Technologies
- React
- Vite
- Tailwind CSS
- Claude AI API
- Lucide Icons
